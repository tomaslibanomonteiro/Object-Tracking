Generates stack of frames and corresponding player map for training data.

This function takes the video cap as input reads several frames downsamples and rescales the frames if applied and converts the frame to grayscale. The output of this function is a stack of np.arrays which contnains the read frames as well as the corresponding times beginning from the start of the video.

Parameters
capinput video capture.
dfdataframe of sensor data
start_frameframe number to start with.
out_fpstarget fps of stack.
stack_sizetarget size of output stack.
data_sizetarget size of data (W,H).
Returns
train_data, train_result - frame stack of video, frame stack of sensor maps.
