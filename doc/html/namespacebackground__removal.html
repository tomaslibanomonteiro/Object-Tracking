Detailed Description

run this file to remove the background: 

python background_removal.py --input FILE_PATH_TO_VIDEO --algo ALGORITHM_NAME

FILE_PATH_TO_VIDEO: usually, this is the video that we want to remove background: data/videos/20220307-1307-214.mp4
ALGORITHM_NAME: 'KNN' or 'MOG2'  

- It will output an mp4 video with no background
- To see the options you can change to remove the background in different ways, read the comments on
    "ARGUMENTS THAT CAN BE CHANGED BELOW" section of the code

Get Foreground of the frame given as input to the function.

This function takes the frame as input and applys the background subtractor to it. After, applys erode, dilation and another erode according to the numbers given as arguments.

Parameters
frameone frame from the video
backSubbackground subtraction method (KNN or MOG2)
first_erodenumber of iterations to erode foreground frame
dilatenumber of iterations to dilate foreground frame
second_erodenumber of iterations to erode foreground frame after dilation
Returns
fgmask foreground frame where algorithm was apllied
